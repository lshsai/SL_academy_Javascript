Background-repeat의 옵션

repeat : default
repeat-x, y : 가로, 세로 반복
no-repeat : 반복 x
space : 반복 할 수 있는 공간만큼
round : 공간만큼(간격 없이) 반복)

Background-position의 옵션

left, right, top, bottom, center 조합가능

Background-attachment의 옵션

fixed : 스크롤 이동해도 고정 scroll : 기본 값

Initial

대부분의 property에 들어갈 수 있는 value
설정을 초기상태로 만들어 줌
background-repeat : initial; (<- 기본으로 돌리기)

background-size 옵션들

auto : 원본 그림 크기
cover : 컨테이너 크기로 조정
contain: 이미지가 모두 보이게 조정
숫자px ; 고정 크기
숫자% ; 컨테이너에서 차지하는 비율
컨테이너? 해당 tag(Element)의 크기
background-size: Apx(%) Ypx(%)
