font-size를 모든 브라우저에서 일괄적으로 맞추기

  1. body의 font-size를 100%로 설정
  2. Element들의 font-size는 em으로 설정
  3. 모든 웹 브라우저에서 같은 크기가 된다
  4. zoom과 resize도 자연스러워진다

Responsive Font Size

브라우저의 크기를 변경하면 글자크기가 어떻게 될까

Viewport는 브라우저의 크기를 말한다.
1vw : Viewport의 1%를 말한다.
ex) Viewport가 50cm일 때, 1vw는 0.5cm
