background-repeat의 옵션들

repeat : 그냥 반복 (default)
repeat-x : 가로로 반복
repeat-y : 세로로 반복
no-repeat : 반복 X
space : 반복 할 수 있는 공간만큼
round : 공간만큼 (간격없이) 반복

background-position 옵션들

left, right, top, bottom, center (조합 가능)

background-attachment 옵션들

fixed : 스크롤을 이동해도 그림위치가 고정됨
scroll : 스크롤됨 (default)

Initial

대부분의 property에 들어갈 수 있는 value
설정을 초기 상태로 만들어준다
예) background-attachement의 initial은 scroll이다

background-size 옵션들

auto : 원본 그림 사이즈
cover : 이미지 크기를 컨테이너 전체를 뒤덮는 크기로 수정해준다
contain : 이미지 크기를 이미지를 모두 볼 수 있게 수정해준다
숫자px : 고정크기 숫자% : 컨테이너에서 차지하는 비율 컨테이너란? background가 속해있는 태그(Element)를 의미한다
